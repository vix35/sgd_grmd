Sistema de gestion de datos

Sistema web para subir datos a base de datos de GRMD

Para subir datos es necesario:

  • Registrarse en el sitio
  • Pedir permisos al administrador segun que tipo de datos se requiera subir
  • Ingresar con cuenta al sitio
  • Elegir una de las opciones de subir datos que aparecen en la barra de navegacion superior
  • Seleccionar el tipo de archivo en el formato que se indica y presionar el boton 'submit'
